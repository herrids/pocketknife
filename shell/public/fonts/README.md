# Fonts

Place Space Grotesk and Space Mono woff2 subsets here.

Download from Google Fonts or use the `gfonts-dl` script:
```
npx google-fonts-dl "Space Grotesk:400,500,600,700" "Space Mono:400,700" --format woff2 -o public/fonts
```

Files expected:
- `SpaceGrotesk-Regular.woff2`
- `SpaceGrotesk-Medium.woff2`
- `SpaceGrotesk-SemiBold.woff2`
- `SpaceGrotesk-Bold.woff2`
- `SpaceMono-Regular.woff2`
- `SpaceMono-Bold.woff2`
